# 🌾 Smart AgroFriend Backend 🌾

Welcome to the **Smart AgroFriend Backend**! This system is designed to revolutionize the agricultural sector by connecting farmers and laborers, offering rental and purchase services for equipment, and providing future crop price predictions using cutting-edge technology.

## ✨ Features

### 🤝 Labor-Farmer Connectivity
- 📝 **Farmers** can post job opportunities.
- 👷 **Laborers** can view and apply for these jobs.
- 💬 **Real-time Chat**: Seamless communication between farmers and laborers.

### 🚜 Rent or Buy Agricultural Equipment
- 🔍 **View** and **book** agricultural equipment for rent or purchase.
- 📦 **Post Equipment** for rent or sale with detailed info and video verification.
- ⭐ **Ratings & Reviews** for equipment and services.

### 🐄 Animal Buy or Sell
- 🐾 **Post Ads** for buying or selling animals.
- 🔎 **Filter & Search** by various criteria.
- 🕒 **Auto-archive** posts after a set period.

### 🌱 Future Crop Price Prediction
- 📊 Predict future prices of crops using **machine learning models** (Linear Regression & LSTM).
- 🥇 Display the **Top 10 profitable crops** based on predictions.

## 🛠️ Technologies Used

- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT), bcrypt
- **File Uploads**: Multer, Cloudinary
- **Environment Variables**: Managed with `dotenv`
- **Real-time Communication**: Socket.IO
- **Machine Learning**: Integrated for crop price predictions
